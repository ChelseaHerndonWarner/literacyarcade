Quick Letter Knowledge Check

Check mixed letter-name knowledge.

Students name each uppercase and lowercase letter once in a mixed order. The timer records how long it takes to complete all 52 letters while the teacher marks missed letters.

Ready to assess

Enter assessment details on the right, print the matching mixed-letter student sheet, then start the timer when the student begins naming letters.

How it works

  1. Print the mixed-letter student sheet.
  2. Start the timer when the student begins.
  3. Tap missed letters on the scoring grid.
  4. Add what the student said when it helps.
  5. Review missed letters and pattern notes.
